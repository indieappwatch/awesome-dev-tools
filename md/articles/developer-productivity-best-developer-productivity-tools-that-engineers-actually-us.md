# Best Developer Productivity Tools That Engineers Actually Use in 2024

> A curated guide covering the best editor tools, platforms, and workflows for modern developers and indie hackers.

The right developer productivity tools can reclaim hours of your week that you'd otherwise spend fighting your setup. After years of experimenting with different editors, CLIs, and workflows, these are the tools that have genuinely changed how developers write, test, and ship code.

## Featured Tools & Platforms


### [VS Code](https://code.visualstudio.com)

The editor that redefined what a code editor could be. Intellisense, a massive extension marketplace, and deep GitHub integration make it the default choice for most developers.

**Highlight**: The most extensible code editor available


### [Zed](https://zed.dev)

A GPU-accelerated editor built by the creators of Atom. Written in Rust for native performance, with first-class collaboration features built in.

**Highlight**: Next-gen editor with native performance


### [Warp](https://warp.dev)

A terminal that reimagines the command line with a modern UI, auto-complete, and workflow blocks. Blocks turn terminal output into interactive elements.

**Highlight**: Terminal with modern UX and AI-powered completion


### [Neovim](https://neovim.io)

The modal editor that developers either love or obsess over. Extensible with Lua, tree-sitter for syntax, and LSP support for intelligent code understanding.

**Highlight**: The extensible modal editor for power users


### [Insomnia](https://insomnia.rest)

A cross-platform API client with a clean interface for designing, debugging, and testing APIs. Supports REST, GraphQL, and gRPC.

**Highlight**: Feature-rich API client for REST and GraphQL


### [Bruno](https://www.usebruno.com)

An open source API client that stores collections as plain text files (git-friendly). Fast, lightweight, and respects your privacy — no cloud account required.

**Highlight**: Open source API client that stores collections in git


### [TablePlus](https://tableplus.com)

A native, lightweight database GUI for macOS and Windows. Clean interface for querying and managing PostgreSQL, MySQL, SQLite, Redis, and more.

**Highlight**: Native database GUI for every major DB


### [DevToys](https://devtoys.app)

A Swiss army knife of utilities for developers: JSON formatter, JWT decoder, hash generator, regex tester, and dozens more — all in a single Windows/macOS app.

**Highlight**: Developer utilities all in one offline app


### [Fig](https://fig.io)

Adds visual autocomplete to your existing terminal. Works with bash, zsh, and fish. Community-built completions for hundreds of CLI tools.

**Highlight**: Visual autocomplete for any terminal


### [Oh My Zsh](https://ohmyz.sh)

A community-driven framework for managing Zsh configuration. Hundreds of plugins and themes that make the terminal more pleasant to use every day.

**Highlight**: The community framework that makes zsh delightful
