# No-Code & Automation

No-code tools have matured beyond simple form builders into platforms capable of supporting real businesses. The visual-first approach has proven itself for a wide range of applications, from internal tools to customer-facing products.

Automation platforms tie these tools together, creating workflows that would otherwise require custom integration code. The combination of visual builders and automation opens up possibilities that previously required engineering resources.

Whether you're validating an idea, building an internal tool, or creating a full product, these no-code and automation tools offer a path forward without writing code.

## Popular Tools in This Category

- **Bubble** — Build products without writing code
- **Zapier** — Connect your apps and automate workflows
- **n8n** — Workflow automation tool with code-first flexibility
- **Notion** — The connected workspace for your wiki, docs, and projects
- **Supabase** — The open source Firebase alternative with a Postgres foundation
