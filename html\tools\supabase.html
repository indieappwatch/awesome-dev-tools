About Supabase

Supabase takes the developer experience that made Firebase popular and reimagines it on top of PostgreSQL, giving you a proper relational database with all the modern conveniences. The combination of a powerful query engine with real-time subscriptions and instant APIs makes building data-driven applications remarkably straightforward.

The Auth system handles everything from email/password to OAuth providers without requiring server-side code, and Row Level Security policies in Postgres give you fine-grained control over who can access what. Authentication state syncs automatically to the client libraries, eliminating the boilerplate that typically comes with user management.

Storage is another area where Supabase shines — file uploads go directly from the browser to your storage bucket, with automatic image transformations and CDN distribution. The dashboard makes database management visual, but everything is also accessible via the CLI and API, so you can automate your workflow however you prefer.

Key Features

  • PostgreSQL database with automatic REST and GraphQL APIs
  • Real-time subscriptions over WebSocket connections
  • Row Level Security for fine-grained access control
  • Authentication with 20+ OAuth providers
  • Edge Functions running on Deno at the edge
  • Storage API with automatic image transformations
  • Database migrations and version control via CLI
  • Multi-region deployments with global edge infrastructure
  • Client libraries for JavaScript, Flutter, Swift, Kotlin, and more
